# YouTube Logo → Subscriptions Userscript

Replace the top-left YouTube logo link so it opens your **Subscriptions feed** instead of the homepage.

Why? I have youtube history disabled, and I want to open my subscriptions feed instead of the homepage when I click the logo.

Heavily inspired by [Scripts.yt](https://scripts.yt).

---

## 📦 Installation

### 1. Install a userscript manager

- Tampermonkey
- Violentmonkey

### 2. Install the script

[![Install Userscript](https://img.shields.io/badge/install-userscript-brightgreen?style=for-the-badge)](https://raw.githubusercontent.com/verylongdelay/youtube-scripts/main/youtube-logo-to-subs.user.js)

Open:

```
https://raw.githubusercontent.com/verylongdelay/youtube-logo-to-subs/main/youtube-logo-to-subs.user.js
```

Your userscript manager will prompt you to install it.

---

## ⚠️ Notes

- Only affects navigation when clicking the logo
- Does **not** modify any other YouTube behavior
- If YouTube significantly changes its header structure, the userscript will need to be updated.
